# Структура проекта

## Основные папки

- `assets/images/logos/`
  Сюда клади логотипы, гербы, постеры и главные изображения проекта.

- `assets/images/buttons/`
  Сюда удобно складывать иконки и картинки для кнопок.

- `assets/images/backgrounds/`
  Сюда клади фоновые изображения, текстуры и декоративные подложки.

- `assets/images/banners/`
  Сюда клади широкие баннеры для шапки сайта, разделов и крупных визуальных блоков.

- `content/laws/`
  Здесь лежат `.txt` файлы со статьями, кодексами и уставами.

- `content/questions/`
  Здесь лежат `.txt` файлы с вопросами для экзаменов и тестов.

## Как добавлять новое

1. Новый логотип:
   Положи файл в `assets/images/logos/`

2. Новую картинку для кнопки:
   Положи файл в `assets/images/buttons/`

3. Новый фон:
   Положи файл в `assets/images/backgrounds/`

4. Новый баннер:
   Положи файл в `assets/images/banners/`

5. Новый закон:
   Положи `.txt` в `content/laws/`

6. Новый файл вопросов:
   Положи `.txt` в `content/questions/`

## Важно

- Если ты просто заменяешь уже используемый файл с тем же именем, код менять не нужно.
- Если добавляешь новый файл с другим именем и хочешь использовать его на сайте, тогда нужно обновить путь в коде.
