# Session Memory

## Что было сделано в этой сессии

- Осмотрена папка проекта и подтверждено наличие `.txt` файлов со статьями и логотипа `sang.png`.
- Проверена кодировка документов.
- Создан стартовый интерфейс сайта для SANG.
- Добавлен просмотрщик документов и поиск по ним.
- Переработан блок тестирования под отдельные темы.
- Добавлены билеты по 10 вопросов.
- Добавлен режим экзамена на 3 билета и 15 минут.
- Добавлена автоматическая генерация вопросов по статьям.
- Подключён ручной файл `ВнутУставВопросы.txt` для темы `Внутренний устав`.
- Добавлено перемешивание вопросов и вариантов ответов.
- Добавлен локальный сервер `server.js` для нормального тестирования через браузер.
- Убран блок `Самый выгодный вариант`.
- Главный экран переделан в официальный стиль `Экзаменация Личного Состава S.A.N.G.`.
- Перед экзаменом теперь требуется ввести имя, фамилию и статик ID.
- После экзамена выводится экзаменационная ведомость с данными и итогом.
- На экзамене действует правило: не более 3 ошибок.
- Убраны автогенерируемые вопросы вида `Как называется статья...`.
- Теперь в экзаменации используются только ручные файлы вопросов.
- При ошибках теперь показываются правильные ответы и ссылка на статью из вопроса.
- В блоке чтения документа сделан отдельный внутренний скролл.
- Создан `README.md` с краткой инструкцией.

## Текущая идея продукта

Это не сложная веб-система, а лёгкий и красивый статический сайт, который:

- можно открыть локально;
- можно бесплатно выложить в интернет;
- можно быстро расширять обычным редактированием `app.js`.
- уже умеет запускать отдельные тематические проверки знаний.

## Что важно помнить дальше

- Пользователь хочет не тратить деньги.
- Сайт должен быть "супер-крутым", но без лишней сложности.
- Основная ценность проекта: тест по законам GTA 5 RP на основе уже существующих текстов.
- Пользователь готов давать тексты со статьями и дополнительный контент.
- Пользователю нравится текущий дизайн.
- Пользователь хочет аккуратные названия тем вместо имён файлов.
- Пользователь хочет билеты и экзамены с конкретными правилами.
- Пользователь добавил готовый файл с вопросами, вариантами и правильными ответами для `Внутреннего устава`.
- Пользователь хочет официальный, презентабельный и военный стиль интерфейса.
- Пользователь не хочет глупые автоген-вопросы по статьям.
- Пользователь удалил старые вопросы внутреннего устава и хочет использовать новый формат файлов вопросов, как в `УКВопросы.txt`.
